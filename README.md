## Описание
Учебный проект по тестированию виджета чат-бота. В приложение на React встроен виджет `@hexlet/chatbot-v2`; тесты проверяют рендер, пользовательские сценарии, интеграцию с хост-приложением и крайние случаи конфигураций шагов.

## Технологии
- React + Vite
- Vitest + React Testing Library (`@testing-library/react` v16, `@testing-library/user-event`, `@testing-library/jest-dom`)
- Чат-бот: `@hexlet/chatbot-v2`

## Команды
- Установка зависимостей: `npm install`
- Запуск dev-сервера: `npm run dev`
- Сборка: `npm run build`
- Тесты: `npm test` (или `npm test -- --run`)
- Линтер: `npm run lint`
